Colocando Links Externos No Meu Site

Para se colocar um link externo em qualquer site, é necessário adicionar a tag <a>. Mas atente-se diego, pois, se apenas colocar essa tag, sua página irá ser sobreposta pelo link que você adicionou.
Mas para que isso não ocorrá, basta adicionar a sua tag <a href ''url'' target=''blank'' rel=''external''>.(Não está totalmente certo de propósito).
